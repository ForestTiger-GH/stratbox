# `stratbox.common`

`stratbox.common` — слой общих утилит, не привязанных к конкретному домену.

Сюда помещается код, который может использоваться одновременно в разных частях библиотеки: банковских формах, архивах, витринах, реестрах или будущих доменах.

## Текущий состав

```text
common/
  time/
    periods.py
```

Основной модуль сейчас — `common.time.periods`.

## Генерация периодов

Модуль `periods.py` содержит универсальные функции генерации опорных дат:

- годы (`Y`);
- кварталы (`Q`);
- месяцы (`M`);
- недели (`W`);
- дни (`D`).

Поддерживаются две точки привязки периода:

- `anchor="start"` — первая дата периода;
- `anchor="end"` — последняя дата периода.

Пример:

```python
from stratbox.common.time.periods import period_points

months = period_points(
    "M",
    date_from="2024-01-01",
    date_to="2024-03-31",
    anchor="start",
)

print(months)
```

Результат:

```text
[datetime.date(2024, 1, 1), datetime.date(2024, 2, 1), datetime.date(2024, 3, 1)]
```

Пример квартальных дат по концу периода:

```python
quarters = period_points(
    "Q",
    date_from="2024-01-01",
    date_to="2024-12-31",
    anchor="end",
)
```

## Использование в доменах

Сейчас генератор периодов используется, в частности, в `macrobanks/cbr_forms` для построения списка отчетных дат.

## Правила расширения

В `common` стоит добавлять только такие функции, которые действительно являются общими. Если логика относится к конкретному предметному процессу, ей место в соответствующем домене, например в `macrobanks/cbr_forms`, `macrobanks/escrow`, `macrobanks/frg` или `macrobanks/cbr_file_collector`.

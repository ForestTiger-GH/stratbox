# rosstat_okved2 — выгрузка ОКВЭД2 (Росстат)

Эта папка содержит три CSV-файла, скачанные с сайта Росстата (ОКВЭД2).

## Что делает код
Модуль `stratbox.registries.rosstat_okved2`:
- находит в папке нужные CSV (meta / structure / data)
- собирает итоговую таблицу:
  - `section` (раздел)
  - `code` (код ОКВЭД2)
  - `name` (наименование)

Использование:
```python
from stratbox.registries import rosstat_okved2
df = rosstat_okved2.read()
```

## Какие файлы должны лежать здесь
Обычно Росстат отдаёт три CSV:
- `meta*.csv` (или `meta.csv`)
- `structure*.csv`
- `data*.csv`

Код распознаёт их по префиксам:
- `meta`
- `structure`
- `data`

## Как обновлять
1) Скачать свежую выгрузку с сайта Росстата (TODO: ссылка будет добавлена).
2) Положить три CSV в эту папку.
3) Можно просто заменить старые файлы.

Важно:
- Не меняйте смысл колонок внутри CSV.
- Если имена файлов отличаются — ничего страшного, главное чтобы начинались с `meta`, `structure`, `data`.

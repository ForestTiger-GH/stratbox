# cbr_replacements — словарь замен (canon / alias)

Это **“человеческий” словарь**, который делает финальный корпоративный канон названий банков.

## Зачем нужно
Даже после нормализации названий могут быть варианты, которые нам неудобны:
- “СБЕРБАНК” → хотим “СБЕР”
- “Т-БАНК” → хотим “ТБАНК”
- “РОССЕЛЬХОЗБАНК” → хотим “РСХБ”
и т.д.

Тут задаётся правило: **какое имя должно быть в итоговых таблицах после нормализации**.

## Формат файла
Файл: `cbr_replacements.csv`

Колонки:
- `canon` — итоговое каноническое имя (CAPS)
- `alias` — имя, которое нужно заменить на `canon` (тоже CAPS)

Пример:
```csv
canon,alias
СБЕР,СБЕРБАНК
ТБАНК,Т-БАНК
РСХБ,РОССЕЛЬХОЗБАНК
```

## Как обновлять (самое простое)
- Открыть `cbr_replacements.csv`
- Добавить новые строки **внизу**
- Сохранить (без изменения заголовка)

Советы:
- Писать в CAPS (как в таблицах внутри компании)
- Можно добавлять сколько угодно `alias` на один `canon`

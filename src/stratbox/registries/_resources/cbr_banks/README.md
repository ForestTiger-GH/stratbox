# cbr_banks — официальный реестр банков ЦБ (XLSX)

В этой папке лежит **официальный XLSX-файл**, скачанный с сайта ЦБ РФ.

## Что делает код
Модуль `stratbox.registries.cbr_banks`:
- читает XLSX из этой папки
- формирует таблицу с полями банка (рег. номер, ОГРН, тип, адрес, статус лицензии и т.п.)
- добавляет нормализованное и каноническое имя банка

Использование:
```python
from stratbox.registries import cbr_banks
df = cbr_banks.read()
```

## Как обновлять
1) Скачать свежий XLSX с сайта ЦБ ([Список кредитных организаций, зарегистрированных на территории Российской Федерации](https://www.cbr.ru/banking_sector/credit/FullCoList/)).
2) Положить файл сюда: `registries/_resources/cbr_banks/`
3) Можно просто **заменить старый файл**, либо положить новый рядом.

Примечание:
- Код берёт **самый свежий .xlsx** в папке.
- Имя файла может быть любым, главное — расширение `.xlsx`.

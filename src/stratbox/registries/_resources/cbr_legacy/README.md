# cbr_legacy — legacy-список банков (обратная совместимость)

Это “старый” фиксированный список банков, который использовался в ранних расчётах (например, в обработке формы 101).

## Зачем нужен
Пока идёт переход на широкий реестр ЦБ и новый канон, legacy-список позволяет:
- не ломать старые отчёты/шаблоны
- воспроизводить “как было раньше” (тот же набор банков и порядок)

## Формат файла
Файл: `cbr_legacy.csv`

Колонки:
- `bank` — короткое имя (как принято в старых отчётах)
- `regn` — регистрационный номер банка
- `sort` — порядок вывода (1..N)

Пример:
```csv
bank,regn,sort
АЛЬФА-БАНК,1326,1
ВТБ,1000,2
...
```

## Как обновлять
- Обычно обновляется редко
- Если нужно:
  - добавляйте строки **внизу**
  - следите, чтобы `sort` был уникальным (1..N)

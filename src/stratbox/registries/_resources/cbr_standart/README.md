# cbr_standart — список “основных банков” (фильтр)

Эта папка содержит список банков, которые считаются “основными” для отдельных отчётов/графиков.

## Как используется в расчётах
В `cbr_banks.read()` формируется колонка:
- `is_canonical` — True/False

Она ставится True для тех банков, которые включены в `cbr_standart.csv` (enabled=True).

Важно:
- перед сравнением названия из `cbr_standart.csv` прогоняются через `normalize_bank_name()`,
  поэтому можно писать “как привыкли” (например, `СБЕРБАНК`) — в расчёте это превратится в “СБЕР”.

## Формат файла
Файл: `cbr_standart.csv`

Колонки:
- `enabled` — True/False
- `bank` — название банка (можно писать по-разному, будет нормализовано)

Пример:
```csv
enabled,bank
True,СБЕРБАНК
True,АЛЬФА-БАНК
True,ВТБ
True,ПСБ
False,КАКОЙ-ТО БАНК
```

## Как обновлять
- Добавлять новые строки **внизу**
- Чтобы исключить банк — поставить `enabled=False`

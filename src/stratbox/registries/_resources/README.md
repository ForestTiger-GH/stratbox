# registries/_resources

Здесь лежат **файлы реестров** (XLSX/CSV), которые используются библиотекой `stratbox`.

## Куда идти дальше
Выбирайте папку по тому, что хотите обновить:

- [`cbr_banks/`](./cbr_banks/) — официальный реестр банков ЦБ (XLSX)
- [`cbr_replacements/`](./cbr_replacements/) — словарь замен (canon/alias) для финального “канона”
- [`cbr_standart/`](./cbr_standart/) — список “основных банков” (фильтр, enabled)
- [`cbr_legacy/`](./cbr_legacy/) — старый список банков для обратной совместимости (bank/regn/sort)
- [`rosstat_okved2/`](./rosstat_okved2/) — выгрузка ОКВЭД2 (3 CSV от Росстата)

## Общие правила обновления (очень просто)
1) Открыть нужную подпапку (см. список выше).
2) Заменить файл (или добавить строки снизу в CSV).
3) Сохранить, закоммитить, запушить в GitHub/GitLab.

Важно:
- Код обычно выбирает “самый свежий” файл (по имени/дате), поэтому **не обязательно переименовывать** файлы, но лучше избегать хаоса:
  - для XLSX: обычно хранится один актуальный файл
  - для CSV: лучше один актуальный файл в папке (или сохранять историю аккуратно)

Каждая подпапка ниже содержит краткую инструкцию “что именно менять”.

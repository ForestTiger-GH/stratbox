# Примеры

Каталог `examples/` содержит примеры использования core-библиотеки.

Там лежат примеры для:

- CBR file collector;
- escrow export;
- FRG stage 1;
- FRG cleanup;
- IO API чтения и записи Excel;
- чтения и записи через pandas.

Примеры не являются частью runtime surface и должны оставаться библиотечными примерами.
